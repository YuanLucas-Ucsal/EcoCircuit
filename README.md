# EcoCircuit
Este trabalho apresenta o desenvolvimento da Ecocircuit, uma plataforma web que visa diminuir impactos ambientais por descarte inapropriado de materiais eletrônico. A EcoCircuit liga as pessoas a pontos de coleta de materiais eletrônicos, para que seja realizado o descarte dos mesmos de forma segura, garantindo menos impacto ao meio ambiente.
